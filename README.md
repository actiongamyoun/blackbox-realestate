# 🏠 부동산 블랙박스 (Real Estate Blackbox)

> **전세사기 제로를 향해** — 중개사 & 임차인 모두를 위한 부동산 계약 안전 플랫폼

---

## 📌 프로젝트 개요

부동산 임대차(전세·월세) 계약 과정에서 발생하는 사기를 **데이터 기반으로 사전 차단**하는 플랫폼입니다.

- 공인중개사의 업무 효율화
- 임차인의 계약 사기 불안 해소
- 객관적 데이터로 위험 요소 시각화

---

## 🖥️ 화면 구성

| 화면 | 경로 | 설명 |
|------|------|------|
| 랜딩 (메인) | `/` | 주소 입력 → AI 분석 시작 |
| 분석 리포트 | `/pages/report.html` | 등기부·위험도·체크리스트·특약 |
| 사후 모니터링 | `/pages/monitoring.html` | 실시간 등기 변동·경매·체납 감지 |
| 중개사 대시보드 | `/pages/dashboard.html` | 매물 관리·계약 파이프라인·고객 관리 |

---

## ✅ 핵심 기능

### 1. AI 매물 분석
- 주소 입력 → 등기부등본 자동 판독 (갑구/을구)
- 전세가율 위험 감지 (KB시세 연동)
- HUG 보증보험 가입 가능 여부 시뮬레이션
- 임대인 국세·지방세 체납 확인 항목

### 2. 위험도 점수화 (0~100)
- 선순위 채권 위험
- 전세가율 위험
- 소유권 변동 이력
- 신탁 등기 여부

### 3. 블랙박스 체크리스트
- 임대인 신분증 대조
- 세금 완납 증명 확인
- 근저당 말소 조건 특약
- 확정일자·전입신고 계획
- HUG 보증보험 가입

### 4. AI 추천 특약 자동 생성
- 위험 요소 기반 필수/권장 특약 자동 생성
- 계약서 삽입 가능한 법적 문구 제공

### 5. 사후 모니터링 (실시간)
- 등기부 변동 즉시 감지 및 알림
- 경매·공매 개시 감지
- 임대인 체납 분기 추적
- 전세가율 변동 추이 차트

### 6. 중개사 대시보드
- 담당 매물 전체 안전 현황
- 계약 진행 파이프라인 (검토→협의→계약→완료)
- 고객별 모니터링 현황
- 매물별 체크리스트 관리
